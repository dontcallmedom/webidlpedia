Definition

Realm execution contexts tied to the LayoutWorklet Global name use LayoutWorkletGlobalScope as a basis for their global object.

This means their global object exposes the following members:


    undefined registerLayout(DOMString name, VoidFunction layoutCtor);

The following interfaces are exposed exclusively in the corresponding realms:

  • BreakToken
  • ChildBreakToken
  • FragmentResult
  • IntrinsicSizes
  • LayoutChild
  • LayoutConstraints
  • LayoutEdges
  • LayoutFragment
  • LayoutWorkletGlobalScope
